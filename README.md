# Tajam - landing page

Tajam is a one-page design template for agencies built with Photoshop. It includes a clean landing page made of sections. Free PSD released by Aji Darmawan.

## Features:

* Responsive design
* Bootstrap Grid
* Mobile Friendly
* Sticky Menu
* Slider
* Contact form

### Plugins:

* Bootstrap Carousel

## Web Development tools:

* jQuery
* Bootstrap
* Sass
* Gulp
